# Mobile Legends Hero Tabs: Obsidia 🎮

A simple, lightweight, and accessible tab UI component built to showcase the marksman hero, **Obsidia**, from Mobile Legends. 

This project is designed as a practical learning experiment to master basic DOM manipulation, event handling, and semantic layout design.

---

## 🚀 Live Demo

You can view the live, interactive project page here:  
👉 **https://github.com/Papa-AR**

---

## ✨ Features

- **Hero-Centric Data:** Includes detailed breakdowns for Obsidia's complete kit:
  * 🔄 **Tab 1:** Passive Ability
  * 🎯 **Tab 2:** First Skill (Damage/Utility)
  * 🛡️ **Tab 3:** Second Skill (Mobility/Defense)
  * 💥 **Tab 4:** Ultimate / Third Skill (Finisher)
- **Comprehensive Mechanics:** Displays crucial gameplay stats like **Skill Descriptions**, **Cooldowns**, and **Mana Costs** dynamically.
- **Accessible Design:** Features standard semantic layout conventions for easy readability.
- **Smooth Interaction:** Snappy transition states between tab panels via clean JavaScript event listeners.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure (`role="tablist"`, buttons, and panels)
- **CSS3:** Responsive layout framing, custom variables, and active states
- **JavaScript (ES6):** Vanilla DOM selection, event indexing, and class toggling
